# Bhajan Clubbing — Next.js Event Landing Page

A stunning, production-grade event landing page for **Bhajan Clubbing — Delhi March 2026** built with Next.js 14, TypeScript, and Tailwind CSS.

## ✦ Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Google Fonts** — Cormorant Garamond, Playfair Display, Space Mono

## ✦ Features

- Animated hero with floating fire particles & mandala background
- Sticky navbar with scroll-aware glass effect
- "What is Bhajan Clubbing" section with animated OM symbol
- Experience grid with hover interactions
- Tiered ticket cards with pricing
- FAQ accordion
- Venue section with details
- Footer with newsletter signup
- Grain texture overlay for premium feel
- Fully responsive (mobile + desktop)
- Custom scrollbar styling
- Gradient text effects

## ✦ Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open in browser
http://localhost:3000
```

## ✦ Build for Production

```bash
npm run build
npm start
```
